🛠 For Developers

This file uses only native browser APIs — no external libraries, no build step. Open browser DevTools Console (F12) to see raw sensor event data being logged.

HTTPS required: All three sensors (geolocation, device orientation, camera) are blocked by browsers on plain HTTP or file:// URLs. Deploy to Vercel/Netlify for a free HTTPS domain, or test on localhost.

iOS 13+ note: Safari requires an explicit permission prompt for DeviceOrientationEvent — this is handled in the code via requestPermission().
